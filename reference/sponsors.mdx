## FAQ

### Can I sponsor anonymously?

Yes. Select "Private sponsor" when signing up on GitHub Sponsors.

### Can I change tiers?

Yes. Change your tier anytime via GitHub Sponsors settings.

### Are sponsorships tax-deductible?

Depends on your jurisdiction. Consult a tax professional. GitHub Sponsors provides receipts.

### Can I sponsor with a one-time payment?

Yes. GitHub Sponsors supports one-time contributions at any amount.

### What happens if I cancel?

You keep benefits until the end of your billing period. No refunds for partial months.

### Do you offer invoices?

Yes. GitHub Sponsors provides invoices for all payments. Contact us for additional documentation.
